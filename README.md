# Anomaly Storyteller

AI-Powered Ads Performance Monitoring & Incident Storytelling Platform

## 🚀 Overview

**Anomaly Storyteller** is an AI-powered monitoring platform designed to help teams detect, analyze, and communicate anomalies in advertising performance automatically.

The system transforms raw performance metrics into structured incident summaries and actionable narratives, helping business, marketing, and operations teams quickly understand what happened, why it matters, and what actions should be taken.

Instead of manually checking dashboards and writing reports, users can input performance data and receive:

* AI-generated anomaly analysis
* Severity classification
* Root cause storytelling
* Business impact explanation
* Recommended actions
* Automated notifications

## 📌 Problem Statement

Monitoring advertising performance manually can be difficult because:

* Performance metrics change rapidly
* Teams receive large amounts of unstructured data
* Incident reports are inconsistent
* Root cause analysis takes time
* Communication between technical and business teams is inefficient

This project solves these problems by automating anomaly detection and transforming data into easy-to-understand business stories.

## 🧠 How It Works

1. Ads performance data is submitted manually or automatically.
2. The system validates and structures the data.
3. AI analyzes anomalies and business metrics.
4. The storytelling engine generates:
   * Incident narrative
   * Severity level
   * Root cause analysis
   * Suggested actions
5. Results are displayed on the dashboard and sent to Telegram/Sheets.
